Zeenea Connector Downloads
Note:
  • Some plugins may require a minimum version of the scanner.
  • The latest version of the scanner can be downloaded from your instance: 
    https://[instance-name].zeenea.app/admin/settings/scanners
  • A plugin can contain several connectors.
  • Templates files to configure each connector are available in this Github repository.
  • For installation instructions, see Zeenea Scanner Setup.
