# Tenant Management

## Supported Languages

Language field can be defined with one of these values:

* **en** (english)
* **fr** (french)
* **de** (german)
* **nl** (dutch)
* **no** (norwegian)
* **es** (spanish)
* **custom_1** (to translate category into data product with a default fallback to **en** for everything else)
