<!-- #p100171 -->
Recommendations for all browsers:

- <!-- #p100177 -->
  For all browsers, you must enable JavaScript, cookies, and TLS 1.0 at least.

- <!-- #p100186 -->
  We recommend a minimum screen resolution of 1440x900 for the best possible user experience.
